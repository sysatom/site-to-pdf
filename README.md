# Site2Pdf

Convert Website to PDF

### Usage

> python site2pdf.py https://docs.python.org/3/tutorial/ tutorial.pdf

### Requirements

- Python 3.7
- [pipenv](https://github.com/pypa/pipenv)
- [wkhtmltopdf](https://wkhtmltopdf.org/)
